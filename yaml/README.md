# YAML Beispiele und Aufgaben

Dieser Ordner enthaelt geloeste YAML-Beispiele aus dem Erklaerblatt.
Die Erklaerungen stehen direkt in den Dateien als englische YAML-Kommentare.

## Dateien

- `01_basic_key_values.yaml` - simple key-value pairs
- `02_person_example.yaml` - basic person data
- `03_nested_server.yaml` - indentation and nested values
- `04_data_types.yaml` - strings, numbers, booleans and null
- `05_simple_lists.yaml` - simple lists
- `06_list_with_properties.yaml` - list items with properties
- `07_multiline_texts.yaml` - multiline text with `|` and `>`
- `08_complete_course_example.yaml` - complete course example
- `09_docker_compose_example.yml` - Docker Compose example
- `10_aufgabe_docker_compose_loesung.yml` - simple solved Docker Compose task
- `11_uebungsblatt_transferaufgaben_loesung.yaml` - solved YAML transfer exercises
- `11_uebungsblatt_transferaufgaben_antworten.md` - written answers for the worksheet questions

## Important YAML Rules

- Use spaces, not tabs.
- Use the format `key: value`.
- Put one space after `:`.
- Keep indentation consistent.
- Use `#` for comments.
