# Hausaufgabe Funktionen Uebung

Dieser Ordner enthaelt die Loesung fuer das Hausaufgabenblatt.

## Datei

- `funktionen_uebung.sh`

## Inhalt

Das Skript enthaelt:

- Funktion `begruessung`
- Funktion `pruefe_benutzer`
- Funktion `pruefe_dateien`
- Funktion `zeige_menue`
- eine `while`-Schleife fuer das Menue
- eine `case`-Auswertung fuer die Auswahl

## Starten

```bash
bash funktionen_uebung.sh
```
